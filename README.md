# Repo Docs + Ontology Skills

This repository packages two complementary Codex skills:

- `repo-docs-intelligence-bootstrap`
- `lightweight-ontology-core`

## Included Skills

### `repo-docs-intelligence-bootstrap`

Use this skill to align repository documentation, `AGENTS.md`, and a minimal intelligence layer with the live codebase.

### `lightweight-ontology-core`

Use this skill to extract and maintain entities, relations, claims, evidence, and segments as a lightweight ontology layer over project documents.

## Suggested Workflow

1. Run `repo-docs-intelligence-bootstrap` first to establish current-state docs and repository guidance.
2. Add `lightweight-ontology-core` when document knowledge needs structured claims, evidence, and retrieval-ready segments.
